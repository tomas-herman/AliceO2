5 # DataBlockBase
6 DataBlockWrapper - wrapper for raw data structures\
7 There should be three static fields in raw data structs, which defines its "signature":\
8  payloadSize - actual payload size per one raw data struct element (can be larger than GBTword size!)\
9  payloadPerGBTword - maximum payload per one GBT word\
10  MaxNelements - maximum number of elements per data block(for header it should be equal to 1)\
11  MinNelements - minimum number of elements per data block(for header it should be equal to 1)\
12 
13 Also it requares several methods:\
14  print() - for printing raw data structs\
15  getIntRec() - for InteractionRecord extraction, should be in Header struct\
16 
17 DataBlockBase - base class for making composition of raw data structures, uses CRTP(static polyporphism)\
18 usage:\
19  class DataBlockOfYourModule: public DataBlockBase< DataBlockOfYourModule, RawHeaderStruct, RawDataStruct ...>\
20  define "deserialization" method with deserialization logic for current DataBlock\
21  define "sanityCheck" method for checking if the DataBlock is correct\
22 Warning! Classes should be simple, without refs and pointers!\
23 Example:
24  /alice/O2/Detectors/FIT/FT0/raw/include/FT0Raw/DataBlockFT0.h
25 # DigitBlockBase
26 Base class for digit block, which containes "digit" structures\
27 usage:\
28  class DigitBlockYourDetector : public DigitBlockBase<DigitBlockYourDetector>\
29  define "template <class DataBlockType> void processDigits(DataBlockType& dataBlock, int linkID)" with Raw2Digit conversion logic\
30  define "template <class DigitType,...>void getDigits(std::vector<DigitType>&... vecDigits) for pushing digits into vectors which will be pushed through DPL channels,
31  variadic arguments are applied\
